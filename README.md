# Titanic Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset.

## Tasks Performed
- Data Loading
- Data Cleaning
- Missing Value Handling
- Outlier Detection and Treatment
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Data Visualization

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Female passengers had higher survival rates.
- Passenger class influenced survival.
- Fare contained significant outliers.
- Missing values were handled appropriately.

## Dataset
Titanic Dataset
